Project Summary

The purpose of this project is to provide various options for our end user to get rating/review based information about food buisnesses in 11 metropolitans area in North America and coffee shops in Atlanta and SFO.

We utilized Yelp API to form our analysis and visuals of coffee shops in Atlanta and San Francisco. The Yelp API was limited on information we could use to create multiple visuals and provide vast options for analysis. As a result we found a Kaggle Yelp dataset which provided the additional information needed.

To create this project we used Python Flask powered restful API, HTML/CSS, JavaScript, MongoDB, Leaflet, Anime JS library and D3.
